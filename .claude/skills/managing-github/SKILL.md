# GitHub操作スキル

## 前提条件

- 環境変数 `GH_TOKEN` が設定されている必要があります
- 必要なコマンド: `curl`, `jq`, `git`

## 操作ガイド

**Issue操作**（取得・作成・更新・コメント・サブIssue）→ [ISSUE-OPERATIONS.md](ISSUE-OPERATIONS.md)を参照
**PR操作**（検索・取得・作成・CI確認・マージ）→ [PR-OPERATIONS.md](PR-OPERATIONS.md)を参照
**スレッド・ワークフロー操作**（一覧・詳細・返信・解決・ログ取得）→ [THREAD-OPERATIONS.md](THREAD-OPERATIONS.md)を参照
**共通処理・参考情報**（複数行テキスト・リポジトリ情報・内部スクリプト・制限事項）→ [COMMON.md](COMMON.md)を参照
