# 品質チェックスキル

以下のコマンドを順番に実行し、すべてエラーがないことを確認する。

## 実行手順

### 1. ビルドチェック

```bash
npm run build
```

ビルドエラーがあれば修正する。

### 2. リントとフォーマットのチェックと自動修正

```bash
npm run lint:fix
```

このコマンドはリントエラーの自動修正とコードのフォーマットを同時に行います。
コマンド実行後もエラーが残っている場合は、手動で修正してください。

### 3. テスト

```bash
npm test
```

## 完了条件

- すべてのコマンドがエラーなく完了すること
- エラーがあった場合は修正し、再度チェックを実行すること
