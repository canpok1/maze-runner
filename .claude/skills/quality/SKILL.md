# 品質チェックスキル

以下のコマンドを順番に実行し、すべてエラーがないことを確認する。

## 実行手順

### 1. ビルドチェック

```bash
npm run build
```

ビルドエラーがあれば修正する。

### 2. リントチェック

```bash
npm run lint
```

リントエラーがあれば修正する。

### 3. フォーマットチェック

```bash
npm run format
```

フォーマットエラーがあれば修正する。

## 完了条件

- すべてのコマンドがエラーなく完了すること
- エラーがあった場合は修正し、再度チェックを実行すること
